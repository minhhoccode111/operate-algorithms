## **Getting Started**

HTTPS

```bash
git clone https://github.com/minhhoccode111/operate-algorithms.git
```

or SSH

```bash
git clone git@github.com:minhhoccode111/operate-algorithms.git
```

then

```bash
cd operate-algorithms
npm install
npm start
```
